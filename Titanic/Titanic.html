The database contains 1309 rows representing each passenger of the Titanic.
The target value "survived" specifies wether the passenger survived or not.
13 features are available giving precision on each passenger' situation.

Categorical Values : 

- pclass : classe of the passenger from in [1,2,3]
- sex : sex of the passenger in [female, male]
- embarked : port of embakation [C for Cherbourg, Q for Queenstown, S = Southampton]
- boat : life boat number (if survived)

Numerical Values : 

- sibsp : number of siblings/spouses of the passenger on the ship [0 to 8]
- parch : number of parents/children of the pasenger on the ship [0 to 6]
- fare : price paid per the passenger
- age : age of the passenger

Unique Values : 

- name : name of the passenger
- ticket : ticket number of the passenger (939 unique values)
- cabin : cabin number of the passenger (186 unique and 1014 null)
- body : body number (if didn't survived)
- home.dest : destination (369 unique values, 564 null)
